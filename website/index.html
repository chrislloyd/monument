Monument is a study in building a programming environment from the ground up for AI. Our current operating systems are based around files, processes, buttons. AI uses tokens, tools and Markdown.

What is a "programming environment"? It's not VSCode. It's closer to LISP machines , the Canon Cat or Dynamicland . Cursor and Copilot are useful additions to today's dead (as in the opposite of live ) programming environments because they only involve editing code and not any actual execution. A programming environment (by my definition) is both live (in that it involves dynamic execution) and malleable (you can edit it while it runs).

If all of this sounds lofty and grand - please be kind and learn from my mistakes. ✌️
